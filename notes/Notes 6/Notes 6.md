# For each of the following commands include: Definition, Usage/Formula, and Examples.

## 1. **mkdir**
Creates a new directory (folder) in Linux.

   **Usage/Formula**
    ```mkdir directory_name```

   **Examples**
    ```mkdir homework```→ Creates a folder named homework
    ```mkdir projects``` → Creates a folder named projects

## 2. **touch**
Creates a new empty file or updates the timestamp of an existing file.

  **Usage/Formula**
    ```touch file_name```

   **Examples**
    ```touch notes.txt``` → Creates an empty file named notes.txt
    ```touch report.doc``` → Creates an empty file named report.doc
   
## 3. **rm**
Removes (deletes) files permanently.

  **Usage/Formula**
    ```rm file_name```

   **Examples**
    ```rm notes.txt``` → Deletes notes.txt
    ```rm oldfile.doc``` → Deletes oldfile.doc
   
## 4. **cp**
Copies files or directories from one location to another.

  **Usage/Formula**
    ```cp source destination```

   **Examples**
    ```cp file1.txt file2.txt``` → Copies file1.txt into file2.txt
    ```cp notes.txt backup.txt``` → Creates a copy named backup.txt
   
## 5. **mv**
Moves files/directories or renames them.

  **Usage/Formula**
    ```mv source destination```

   **Examples**
    ```mv old.txt new.txt``` → Renames old.txt to new.txt
    ```mv notes.txt homework/``` → Moves notes.txt into homework folder
